#TesloDB API

1. clonar project
2. ```npm  install``` 
3. clonar el archivo ```.env.template``` y renomblarlo a ```.env```
4. cambiar las variables de datos 
5. Levantar la base de datos 
docker-compose up -d
6. ejecutar el Seed 
```
http://localhost:3000/api/seed
```
